# vitaminjs-cli [![NPM version][npm-image]][npm-url]

cli for [vitaminjs](https://npmjs.com/package/vitaminjs)

[![Dependency Status][daviddm-image]][daviddm-url]

## Install

```sh
npm install -g --save vitaminjs-cli
```

## Usage

```sh
vitaminjs --help
```
